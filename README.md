# 🍽 Restaurant Manager
Sistema de gestión para restaurantes desarrollado en equipo.
## 📋 Features
- [ ] Gestión de menú (agregar/remover items)
- [ ] Procesamiento de órdenes
- [ ] Sistema de reservas
